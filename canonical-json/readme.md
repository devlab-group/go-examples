# Canonical json

Illustrates how to convert JSON string into canonical format

Before use install `github.com/ugorji/go/codec` library:

```shell
go get github.com/ugorji/go/codec
# Or
dep ensure
```

Or with bake

```shell
bake go get github.com/ugorji/go/codec
# Or
bake dep ensure
```
